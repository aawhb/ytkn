# Code of Conduct

## Our Pledge

We want this project to be a respectful and useful place for Obsidian users and contributors.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but accept corrections.
- Keep issues and pull requests focused on the plugin.
- Avoid sharing private vault content, API keys, or personal data.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing someone else's private information.
- Repeated off-topic or disruptive comments.

## Enforcement

The maintainer may edit, hide, or remove comments and may block users who violate this code of conduct.
